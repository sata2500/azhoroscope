## v1.0 - MVP

**Özellikler:**

1. Kullanıcı kaydı ve giriş (Google OAuth)
2. Günlük burç yorumları (12 burç)
3. Haftalık burç yorumları
4. Aylık burç yorumları
5. Doğum haritası oluşturma
6. Doğum haritası AI analizi
7. Admin paneli - AI model seçimi (Gemini modelleri arasından)

9. Admin paneli - API key yönetimi
10. Admin paneli - Model parametreleri (temperature, max_tokens vb.)
